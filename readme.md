# KFT Automation

A Node.js-based test automation framework using Playwright and custom API utilities.

## 🚀 Project Overview
This repository contains scripts and utilities for automating mobile and web workflows, API testing, and token generation used across various services such as angelOne and muthoot. The project leverages Playwright for UI tests and custom Node modules for HTTP requests, payload management, and mobile driver control.

## 🛠️ Getting Started

### Prerequisites
- Node.js (>= 16.x)
- npm or yarn
- [Playwright](https://playwright.dev/) installed globally or via project dependency

### Installation

```bash
git clone https://github.com/bonthusrihari-collab/kft-automation.git
cd kft-automation
npm install
# or yarn install
```

### Running Tests
- **Playwright UI tests:**
  ```bash
  npx playwright test
  ```
- **Specific test file:**
  ```bash
  npx playwright test src/tests/playwright/example.spec.js
  ```

### Generating Tokens / API Scripts
The `src/tests/angelOne/generateToken.js` script and others can be executed via node directly:
```bash
node src/tests/angelOne/generateToken.js
```

## ⚙️ Configuration
- **URLs:** `src/config/urls.js`
- **Mobile capabilities:** `src/config/mobileCaps.js`
- **Environment variables:** Use `.env` if needed and load via your preferred library.
